# PyQtWeather
Приложение в рамках проекта в УрФУ по PyQt

Вписывайте город, и получаете текущую погоду! Все запросы сохраняются в историю, при перезапуске не исчезают.

## Стек: 
1. Python
2. PyQt
3. SQLite
4. Open weather map API
